A multi-modal chatbot powered by Groq, Deepgram, and Flux — built to go beyond text-only conversations.

## Features

- **Fast, contextual chat** — powered by Groq for near-instant responses
- **Speech to text** — talk to Askbot naturally and have it transcribed in real time
- **Text to speech** — hear responses read back with natural-sounding voice output
- **Image generation** — describe an image in words and Flux generates it
- **Image analysis** — upload a photo and ask Askbot questions about what's in it
